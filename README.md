# 🚀 Raj Technologies
A premium, high-performance corporate landing page built for top-tier Web Development, UI/UX Design, SEO, and Hosting services.

## 📝 Overview
The Raj Technologies platform is designed as the ultimate digital storefront. It leverages the speed of Vite and React combined with fluid animations from Framer Motion to deliver a highly engaging, professional user experience that converts visitors into enterprise clients.

## ✨ Key Features
- **Fluid UI Animations**: Powered by Framer Motion for scroll reveals and layout transitions.
- **Modern Iconography**: Implements `lucide-react` for crisp, scalable vector graphics.
- **Lightning Fast Performance**: Vite implementation ensures near-instantaneous load times.
- **Scalable Architecture**: Highly modular React component structure perfectly suited for enterprise upgrades.

## 🛠 Tech Stack
- Framework: `React 19`
- Bundler/Dev Server: `Vite`
- Animations: `Framer Motion`
- Icons: `Lucide React`

## 🚀 Getting Started

```bash
# Navigate to the project directory
cd raj-technologies

# Install all necessary React dependencies
npm install

# Start the Vite development server 
npm run dev

# Build for production deployment
npm run build
```


## 🌐 Deployment

### Vercel / Netlify (Recommended)
1. Push this repository to GitHub.
2. Connect your GitHub account to [Vercel](https://vercel.com) or [Netlify](https://www.netlify.com).
3. Select this project and use the following settings:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`

### GitHub Pages
1. Install the gh-pages package: `npm install gh-pages --save-dev`
2. Add deployment scripts to your `package.json`.
3. Run `npm run deploy`.
